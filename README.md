# My First Website

这是一个正在持续完善中的个人主页项目，也是我从 0 开始学习网页开发、Git 和 GitHub Pages 的实践仓库。

## 当前页面

- `index.html`：主页
- `process.html`：建站过程记录页
- `project-website.html`：第一个网页仓库项目介绍页
- `roadmap.html`：未来计划页
- `downloads.html`：资料下载中心页

## 重要记录文件

- `BUILD-LOG.md`：记录全过程
- `HANDOFF.md`：记录当前进度、下次待办和每次继续开发前应先看什么

## 如何查看

1. 直接双击 `index.html`
2. 或者访问线上地址：`https://aaahuihui.github.io/my-first-website/`

## 当前资料存储原则

- 网站仓库主要保存网页代码、说明和下载入口
- 大文件如压缩包、PDF、PPT 等建议长期保存在云盘或其他稳定存储中
- 网站中的“资料下载中心”页面负责展示这些资料入口

## 协作约定

- 持续记录 `BUILD-LOG.md`
- 重要阶段维护 `README.md`
- 保留 `HANDOFF.md` 方便以后追溯和继续开发
- 每完成一小步尽量做一次 Git 提交
- 每个阶段完成后继续推送 GitHub
- 不到万不得已，不往 `C 盘` 安装软件
